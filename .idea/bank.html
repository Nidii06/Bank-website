GitHub Guide & Bank App

How to Connect Your Project to GitHub

1

Install Git

Download and install Git from git-scm.com

2

Create a GitHub Account

Sign up at github.com if you don't have one

3

Create a New Repository

Click the "+" icon in the top right and select "New repository"

Tip

Don't initialize with README if you already have a project

4

Initialize Git in Your Project

Open terminal in your project folder and run:

git init
5

Add Your Files to Staging

git add .

This adds all files in the current directory

6

Commit Your Files

git commit -m "First commit"
7

Add the Remote Repository

Copy the URL of your GitHub repository and run:

git remote add origin https://github.com/Nidii06/Bank-website.git
8

Push Your Code to GitHub

git push -u origin master

For main branch (if you're using main instead of master):

git push -u origin main

GitHub Tips

  • Use descriptive commit messages
  • Create branches for new features
  • Regularly pull changes from remote
  • Use .gitignore to exclude unnecessary files
  • Explore GitHub Actions for CI/CD
